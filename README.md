# The Antagonist

Adds a custom AI storyteller, **The Antagonist**, to RimWorld's Choose AI Storyteller screen.

For now it is a clone of Cassandra Classic (identical event pacing) with a placeholder portrait — a starting point for a bespoke storyteller. Pure-XML `StorytellerDef` inheriting `BaseStoryteller`; swap the portrait textures in `Textures/UI/HeroArt/Storytellers/` and tune the `comps` later.

---

A RimWorld 1.6 mod by wishRobber.
